🧹 If something looks wrong 🧹 Si algo se ve mal 🧹 Wenn etwas falsch aussieht

Clear your cache Limpia tu caché Cache leeren

This applies to the generated preview sites, not to Telegram itself. Esto aplica a los sitios de vista previa generados, no a Telegram. Das gilt für die generierten Vorschau-Seiten, nicht für Telegram.

🤖 Android Chrome Android Chrome Android Chrome ▼
  1. Open Chrome → ⋮ menu → Settings → Privacy → Clear browsing data
  2. Abre Chrome → ⋮ menú → Ajustes → Privacidad → Borrar datos
  3. Chrome → ⋮ Menü → Einstellungen → Datenschutz → Browserdaten löschen
  4. Select Cached images + Cookies → Clear
  5. Selecciona Caché + Cookies → Borrar
  6. Wähle Cache + Cookies → Löschen
🍎 iOS Safari iOS Safari iOS Safari ▼
  1. Settings → Apps → Safari → Clear History and Website Data
  2. Ajustes → Apps → Safari → Borrar historial y datos
  3. Einstellungen → Apps → Safari → Verlauf und Daten löschen
🖥️ Desktop Chrome / Brave Chrome / Brave escritorio Desktop Chrome / Brave ▼
  1. Press F12 → Application → Storage → Clear site data → Reload
  2. F12 → Application → Storage → Clear site data → Recargar
  3. F12 → Anwendung → Speicher → Website-Daten löschen → Neu laden
