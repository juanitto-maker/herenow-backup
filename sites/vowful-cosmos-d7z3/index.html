🧹Obtener la última versión

La app es una PWA con caché agresiva — limpia si ves errores

💡 ¿Cuándo hacerlo?Si la app parece desactualizada, no carga bien o muestra errores raros — borra los datos del sitio y recarga.
🤖 Android Chrome
  1. Chrome → ⋮ menú → Ajustes → Privacidad y seguridad → Borrar datos
  2. Selecciona Caché + Cookies y datos del sitio → borrar
  3. Recarga love-me-again.vercel.app

Más rápido: Mantén pulsado el icono de la app → Info app → Almacenamiento → borrar caché y datos.

🍎 iOS Safari
  1. Ajustes → Apps → Safari → Borrar historial y datos de sitios web
  2. Reabrir Safari → love-me-again.vercel.app
🖥️ Chrome / Brave (escritorio)
  1. F12 → pestaña Application → Storage → Clear site data
  2. Recargar la página

Atajo: Ctrl+Shift+R (recarga forzada)

🦊 Firefox
  1. 🔒 candado → Borrar cookies y datos del sitio… → confirmar
  2. Recargar la página

🧹Getting the latest version

The app is a PWA with aggressive caching — clear it if you see issues

💡 When to do thisIf the app looks outdated, won't load properly, or shows weird errors — clear site data and reload.
🤖 Android Chrome
  1. Chrome → ⋮ menu → Settings → Privacy and security → Clear browsing data
  2. Select Cached images and files + Cookies and site data → clear
  3. Reload love-me-again.vercel.app

Faster: Long-press the app icon → App info → Storage → clear cache and data.

🍎 iOS Safari
  1. Settings → Apps → Safari → Clear History and Website Data
  2. Reopen Safari → love-me-again.vercel.app
🖥️ Chrome / Brave (desktop)
  1. F12 → Application tab → Storage → Clear site data
  2. Reload the page

Shortcut: Ctrl+Shift+R (force reload)

🦊 Firefox
  1. 🔒 lock icon → Clear Cookies and Site Data… → confirm
  2. Reload the page
