🧹 Getting the latest version
💡 When to do thisIf the app looks outdated or shows errors — clear the site data. The app uses a service worker that caches files aggressively.
⚠️ Save your credit token first!If you purchased credits, your token is stored in your browser. Clearing site data will delete it. Note it down or use the email receipt before proceeding.
🤖 Android Chrome▼
  1. Open Chrome → go to valor-app.netlify.app
  2. Tap ⋮ menu → Settings → Privacy and security → Clear browsing data
  3. Select Cached images and files + Cookies and site data → Clear data
  4. Return to the app and reload.
Faster: Long-press app icon → App info → Storage → Clear cache + Clear data.
🍎 iOS Safari▼
  1. Settings → Apps → Safari → Clear History and Website Data
  2. Reopen Safari → go to valor-app.netlify.app
Site-only: Settings → Safari → Advanced → Website Data → search "valor-app" → swipe to delete.
🖥️ Desktop Chrome / Brave▼
  1. Press F12 → Application tab → Storage → Clear site data
  2. Reload the page.
Quick reload: Ctrl+Shift+R
🦊 Desktop Firefox▼
  1. Click the 🔒 lock icon → Clear Cookies and Site Data… → confirm and reload.
